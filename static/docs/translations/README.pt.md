## 📖 Introdução

Este é um jogo de memória simples feito com SvelteKit e TypeScript.

<!-- Seção de Recursos -->

## 🚀 Recursos

- Combinação de cartas com a mesma imagem
- Opção de pausa
- Opção de reiniciar
- Cronômetro
- Tentativas
